CSS

Explicando como funciona

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar páginas HTML. Ele controla cores, fontes, espaçamentos e layouts.

Sem CSS, os sites seriam apenas estruturas simples e sem estilo.

Seletores CSS

Os seletores definem quais elementos serão estilizados.

  • * → Seleciona todos os elementos
  • p → Seleciona todas as tags <p>
  • .classe → Seleciona elementos com classe
  • #id → Seleciona elemento com ID específico
  • div p → Seleciona parágrafos dentro de div
  • a:hover → Quando o mouse passa sobre o link

Exemplo:

p { color: red; }
.classe { font-size: 20px; }
#id { background: black; }

Propriedades e Valores

No CSS, cada estilo é formado por uma propriedade e um valor.

Estrutura:

propriedade: valor;

Exemplos:

  • color: blue; → muda a cor do texto
  • font-size: 18px; → muda o tamanho da fonte
  • background: black; → muda o fundo
  • margin: 10px; → espaço externo
